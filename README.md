# sentiment-analysis-project

NLP Sentiment Analysis Project

This project implements Sentiment Analysis using Natural Language Processing (NLP) techniques to classify text as Positive, Negative, or Neutral.
It is suitable for analyzing movie reviews, product feedback, tweets, and user comments.

Features

Text preprocessing using NLP techniques

Sentiment classification (Positive / Negative / Neutral)

Implemented using Python

Uses NLTK / Scikit-learn 

Easy to run and understand

Technologies Used

1.Python 3.x

2.Natural Language Toolkit (NLTK)

3.Scikit-learn

4.Pandas

5.NumPy
